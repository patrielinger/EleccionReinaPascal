⚙️ Checklist de conexión:

  • El otro dispositivo está conectado a la misma red WiFi que esta máquina
  • El puerto 3000 no está bloqueado por firewall
  • Abre el navegador en el otro dispositivo e ingresa la URL arriba
  • Si no funciona, reinicia el router WiFi

⚠️ Solucionar problemas:

  1. Verifica la red: Ambos dispositivos deben estar en el mismo WiFi
  2. Comprueba el firewall: En Windows/Linux, permite el puerto 3000
  3. Prueba con hotspot: Conecta el otro dispositivo al hotspot del teléfono y prueba
  4. Reinicia el servidor: Mata y reinicia el servidor Python
